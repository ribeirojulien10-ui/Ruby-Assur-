# Ruby Assur’ - Codex

Site vitrine local créé from scratch pour Ruby Assur’.

## Lancer le projet

```bash
npm install
npm run dev
```

Le site est prévu pour être consulté en local sur :

```text
http://localhost:3000
```

Le formulaire et le chatbot sont en mode démonstration. Aucun email n’est envoyé, aucun appel externe n’est effectué et aucune clé API n’est nécessaire.
